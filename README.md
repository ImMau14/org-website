## 🚀 Getting Started

#### Prerequisites
- Node.js 18+
- npm, bun or yarn

#### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/bufferring/org-website.git
   cd org-website
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

4. **Start the application**
   ```bash
   # Development
   npm run dev
   
   # Production
   npm start
   ```
